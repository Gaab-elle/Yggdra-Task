# Vídeo de Demonstração

Este diretório deve conter o arquivo `demonstracao.mp4` com:

- **Duração máxima**: 8 minutos
- **Conteúdo**: Explicação clara de todas as etapas e funcionalidades realizadas no projeto
- **Formato**: MP4

## Funcionalidades a serem demonstradas:

1. **Sistema de Autenticação**
   - Login tradicional
   - Login com Google OAuth

2. **Gestão de Tarefas**
   - Criação de tarefas
   - Edição e exclusão
   - Sistema de prioridades
   - Categorização

3. **Interface Kanban**
   - Drag & drop
   - Mudança de status
   - Organização visual

4. **Sistema de Feriados**
   - Verificação automática
   - Snackbar de notificação
   - Integração com API

5. **Notificações**
   - Notificações em tempo real
   - Sistema de alertas
   - Histórico de atividades

6. **Dashboard**
   - Métricas em tempo real
   - Gráficos de produtividade
   - Relatórios

7. **Responsividade**
   - Interface mobile
   - Adaptação para diferentes telas
